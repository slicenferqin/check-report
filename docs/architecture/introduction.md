# Introduction

本文档概述了"检测报告在线查询系统"的完整全栈架构，包括后端系统、前端实现及其集成方案。它作为 AI 驱动开发的唯一真实来源，确保整个技术栈的一致性。

这种统一的方法结合了传统上独立的后端和前端架构文档，简化了现代全栈应用的开发流程，因为这些关注点越来越紧密地交织在一起。
